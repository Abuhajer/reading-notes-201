
# FORMS
### In addition to enabling users to search, forms also allow users to perform other functions online.


## Form Controls
- Text Input Controls
- Checkboxes Controls
- Radio Box Controls
- Select Box Controls
- File Select boxes
- Hidden Controls
- Clickable Buttons
- Submit and Reset Button
